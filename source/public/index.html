Задание

    Нужно разработать таблицу в формате Single Page Application.

    Требования к таблице:

    1. Таблица должна содержать 4 колонки: a) Дата, b) Название, c) Количество, d) Расстояние.
    2. База данных любая. Данные должны приходить в таблицу отсортированные по возрастанию даты.
    3. Таблица должна иметь выбор сортировки по всем полям кроме даты:

      4. a) Выбор колонки, по которой будет фильтрация

      b) Выбор условия (равно, содержать, больше, меньше)

      c) Поле для ввода значения для фильтрации

    4. Фильтрация должна быть в виде двух выпадающих списков и текстового поля.
    5. Вся таблица должна работать без перезагрузки страницы.
    6. Реализация на Vue3.0
